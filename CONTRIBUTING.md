# Contributing to Egov.Integrations.MSign.Soap

Thank you for your interest in contributing to Egov.Integrations.MSign.Soap! We welcome contributions from the community.

## How to Contribute

1. **Report Bugs**: Use the issue tracker to report any bugs or issues you encounter.
2. **Suggest Enhancements**: Suggest new features or improvements by opening an issue.
3. **Submit Pull Requests**:
   - Fork the repository.
   - Create a new branch for your changes.
   - Ensure your code follows the existing style and is well-documented.
   - Submit a pull request with a clear description of your changes.

## Development Setup

- Ensure you have the .NET SDK installed.
- Open the solution `Egov.Integrations.MSign.Soap.sln` in your preferred IDE.
- Run tests to ensure everything is working as expected.
